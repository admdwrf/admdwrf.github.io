    Introduction¶

    I need to had a hard drive on my workstation. I want encrypt data and add read & write cache.

    Operation¶

    SSD¶

    I'll create two partitions:

    • One for the write cache (log0)
    • One for the read cache (cache0)
    gpart add -t freebsd-zfs -b 2048 -a 4k -l log0 -s 32G ada0
gpart add -t freebsd-zfs -a 4k -l cache0 ada0

    Look how it is:

    gpart show /dev/ada0
=>       40  234441568  ada0  GPT  (112G)
         40       2008        - free -  (1.0M)
       2048   67108864     1  freebsd-zfs  (32G)
   67110912  167330688     2  freebsd-zfs  (80G)
  234441600          8        - free -  (4.0K)

    Zpool on the hard drive¶

    gpart create -s gpt ada3
gpart add -t freebsd-zfs -l volume-zdata0 ada3
geli init -l 256 /dev/gpt/volume-zdata0
geli attach /dev/gpt/volume-zdata0
zpool create -m none zdata0 /dev/gpt/volume-zdata0.eli

    Now we'll add write and read cache

    zpool add zdata0 log gpt/log0
zpool add zdata0 cache gpt/cache0

    Quand le sys reboot, il faut redonner la passphrase de chiffrement et faire un zpool import

    geli attach /dev/gpt/volume-zdata0
zpool import
zfs mount
