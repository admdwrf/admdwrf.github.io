    Install FreeBSD on Dell T5000¶

    Introduction¶

    Two months ago, I bought a Dell Precision T5500 Workstation on ebay. For less than two hundred euros, I had a Xeon hexa core, twelve giga byte of RAM, a 250 gb hard drive and a Nvidia Quattro graphic cards. I added to it a PCI-e SATAIII card, two SSD ( one of 128 GB and another of 250 Gb), one hard drive of 1 TB, a wifi pci card, and I replaced the graphic card with an Nvidia GeForce GT 710. Now it's time to install and configure FreeBSD on it.

    Note

    For info, big thank you to Olivier Cochard-Labbé and Allison Nicole Reid for all good informations I found on their websites http://olivier.cochard.me/bidouillage/installation-et-configuration-de-freebsd-comme-poste-de-travail https://cooltrainer.org/a-freebsd-desktop-howto/

    Locales¶

    Put this two lines into the default section in /etc/login.conf:

    ** :charset=UTF-8:\ :lang=en_US.UTF-8:\ **

    Rebuild the login database:

    sudo cap_mkdb /etc/login.conf

    Modify /etc/profile

    vi /etc/profile

LANG=en_US.UTF-8;   export LANG
CHARSET=UTF-8;  export CHARSET

    Set bash at default shell for fralix account¶

    sudo chsh -s /usr/local/bin/bash fralix

    Configure /etc/sysctl.conf¶

    sudo cat >> /etc/sysctl.conf <<EOF
kern.vt.enable_bell=0
hw.syscons.bell=0
kern.ipc.shmmax=67108864
kern.ipc.shmall=32768
kern.sched.preempt_thresh=224
kern.maxfiles=200000
hw.syscons.bell=0
kern.ipc.shm_allow_removed=1
hw.snd.default_unit=0
EOF

    Configure /boot/loader.conf¶

    sudo sysrc -f /boot/loader.conf autoboot_delay=2
sudo sysrc -f /boot/loader.conf kern.ipc.shmseg=1024
sudo sysrc -f /boot/loader.conf kern.ipc.shmmni=1024
sudo sysrc -f /boot/loader.conf kern.maxproc=100000
sudo sysrc -f /boot/loader.conf coretemp_load="YES"
sudo sysrc -f /boot/loader.conf tmpfs_load="YES"
sudo sysrc -f /boot/loader.conf aio_load="YES"
sudo sysrc -f /boot/loader.conf snd_driver_load="YES"
sudo sysrc -f /boot/loader.conf fusefs_load="YES"

    Configure /etc/rc.conf¶

    sudo sysrc ntpd_sync_on_start=yes
sudo sysrc kld_list+=smbus
sudo sysrc microcode_update_enable=YES
sudo sysrc dbus_enable=YES
sudo sysrc hald_enable=YES
sudo sysrc cupsd_enable=YES
sudo sysrc gdm_enable="YES"
sudo sysrc gnome_enable="YES"
sudo sysrc fusefs_enable=YES
sudo sysrc autofs_enable=YES
sudo sysrc automount_enable=YES
sudo sysrc automountd_enable=YES
sudo sysrc autounmountd_enable=YES

    Configure /procfs¶

    sudo vi /etc/fstab

proc           /proc       procfs  rw  0   0

    Configure automount¶

    sudo sed -i "" -e "s/#\/media/\/media/" /etc/auto_master

sudo cat >>/etc/devd.conf <<EOF
# Discard autofs caches, useful for the -media special map.
notify 100 {
        match "system" "GEOM";
        match "subsystem" "DEV";
        action "/usr/sbin/automount -c";
};
EOF

    Xorg¶

    Download the latest Nvidia driver at http://www.nvidia.com and put the archive NVIDIA-FreeBSD-x86_64-384.59.tar.gz to /var/tmp

    Extract the archive and compile/install the nvidia xorg driver

    cd /var/tmp
tar -zxf NVIDIA-FreeBSD-x86_64-384.59.tar.gz
cd NVIDIA-FreeBSD-x86_64-384.59
sudo make install -j4

    Configure xorg:

    sudo cat > /usr/local/etc/X11/xorg.conf.d/20-video.conf <<EOF

Section "Device"

        Identifier "NVIDIA Card"

        VendorName "NVIDIA Corporation"

        Driver "nvidia"

EndSection

Section "Module"
        Load           "freetype"
        Load           "bitmap"
        Load           "type1"
        Load           "glx"
EndSection
